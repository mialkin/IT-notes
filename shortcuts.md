## Terminal

| Shortcut    | Description                 |
| ----------- | --------------------------- |
| Control + F | Move forward one character  |
| Control + B | Move backward one character |
| Control + W | Remove word backward        |
| Control + K | Delete to end of line       |
| Control + Y | Paste                       |
